Если нужны выпадающие меню, их можно создать при помощи a Spry menu — мини-приложения menu из Adobe Exchange или ряда других инструментов javascript или CSS.

Если нужна навигация вдоль верха, просто перенесите ul.nav в верх страницы и заново создайте стиль.

Test

1.Downloaded VSFLEX3.OCX to your system32 folder within the Windows folder. [c:WINDOWSsystem32]
2.Click on the Start button, then click the Run... option
3.In the "Open" line, type the following (without quotes) "regsvr32 VSFLEX3.OCX", then click the "OK" button.
4.If the installation was successful, you should see a message saying that your file was successfully installed/uninstalled.
